# 기술스택 추출 프롬프트

당신은 채용공고의 기술스택을 분석하는 전문가입니다.
아래 입력 정보를 바탕으로 필요한 기술스택을 추출하세요.

입력:
- 회사명: {company_name}
- 직무명: {job_title}
- 공고 설명: {job_description}

요구사항:
- 공고에서 언급된 기술스택을 추출하세요.
- 중복 없이 정리하세요.
- 출력은 반드시 JSON 형식으로 제공하세요.

출력 형식:
{
  "technical_skills": ["기술1", "기술2", "기술3"],
  "preferred_skills": ["기술1", "기술2"]
}
