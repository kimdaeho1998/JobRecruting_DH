# 프로젝트 개요

## 서비스 소개

JobInsight AI는 채용공고 탐색, AI 분석, 비교, 북마크, 지원현황 관리를 하나의 워크스페이스에서 제공하는 통합 채용공고 분석 플랫폼입니다. 사용자는 여러 채용 플랫폼의 공고를 같은 기준으로 살펴보고, AI가 정리한 핵심 정보와 개인 지원 상태를 함께 확인할 수 있습니다.

현재 구현은 MVP 구조입니다. 데이터는 시드/Mock 기반으로 시작하며, 백엔드 API, 프론트엔드 화면, AI 호출 구조, 크롤러 확장 구조를 먼저 구성했습니다. 이후 실제 채용 사이트 크롤링과 사용자별 개인화 분석으로 확장할 수 있도록 설계했습니다.

## 문제 정의

채용 준비 과정에서 반복적으로 발생하는 문제는 다음과 같습니다.

- 공고가 여러 사이트에 흩어져 있어 탐색과 비교가 어렵다.
- 회사마다 공고 작성 방식이 달라 핵심 업무, 요구 기술, 우대 조건을 빠르게 파악하기 어렵다.
- 관심 공고, 지원 여부, 전형 상태를 별도 도구로 관리해야 한다.
- 여러 공고 중 어떤 공고가 본인의 이력과 더 맞는지 판단하기 어렵다.

JobInsight AI는 공고 데이터를 공통 스키마로 정리하고, AI 분석 결과와 지원 상태 관리를 한 화면 흐름에 연결해 이 문제를 줄이는 것을 목표로 합니다.

## 목표 사용자

- 여러 채용 사이트를 동시에 탐색하는 구직자
- 지원 우선순위를 체계적으로 정하고 싶은 개발자
- 공고의 기술스택과 요구사항을 빠르게 비교하고 싶은 사용자
- 지원 현황을 개인 파이프라인처럼 관리하고 싶은 사용자

## 핵심 사용자 흐름

1. 사용자가 공고 목록에서 키워드, 회사, 직무, 지역, 경력, 기술스택으로 검색한다.
2. 관심 있는 공고를 상세 화면에서 확인한다.
3. AI 요약, 기술스택 추출, 적합도 분석을 참고한다.
4. 공고를 북마크하거나 비교 목록에 추가한다.
5. 지원 상태를 저장하고 Application Board에서 관리한다.
6. Dashboard에서 전체 지원 흐름과 공고 분포를 확인한다.

## 프로젝트 차별점

- 단순 공고 목록이 아니라 지원 준비의 전체 흐름을 다룬다.
- AI 프롬프트를 코드와 분리해 유지보수성을 높였다.
- Mock 모드와 실제 API 모드를 환경변수로 전환해 개발과 운영 흐름을 분리했다.
- 크롤러를 사이트별 Provider로 분리해 확장 가능하게 설계했다.
- 백엔드는 API, Service, Repository, Schema, Model 계층으로 책임을 분리했다.
