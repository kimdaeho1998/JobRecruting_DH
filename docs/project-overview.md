# Project Overview

## 1. 서비스 개요
JobInsight AI는 여러 채용 플랫폼의 공고를 하나의 흐름으로 수집·정규화·분석해, 사용자가 보다 쉽게 채용 공고를 비교하고 지원 상태를 관리할 수 있도록 지원하는 AI 기반 통합 채용공고 분석 플랫폼이다.

초기 버전은 크롤링 기능 없이 Mock Data 기반으로 운영되며, 사용자가 공고 요약, 기술스택 추출, 적합도 분석, 공고 비교, 지원현황 관리를 확인할 수 있는 MVP를 목표로 한다.

## 2. 문제 정의
- 채용 공고가 여러 사이트에 분산되어 있어 한 번에 비교하기 어렵다.
- 공고마다 표현 방식과 형식이 다르기 때문에 정보 파악 비용이 높다.
- 지원 현황을 개인이 관리하기 위해 별도 도구나 스프레드시트를 자주 사용해야 한다.
- 기술스택, 경력, 우대조건 등 핵심 정보를 빠르게 파악하기 어렵다.

## 3. 사용자 서비스 흐름
1. 사용자가 웹 서비스에 접속한다.
2. 대시보드에서 Mock Data 기반 공고 목록을 확인한다.
3. 공고를 선택하면 요약, 기술스택, 적합도 분석 결과를 확인한다.
4. 비교 페이지에서 여러 공고를 나란히 비교한다.
5. 지원현황 관리 페이지에서 지원 상태를 업데이트한다.
6. 사용자는 결과를 바탕으로 지원 우선순위를 결정한다.

## 4. MVP 기능 범위
- 공고 목록 조회
- 공고 상세 조회
- AI 기반 공고 요약
- 기술스택 추출
- 적합도 분석
- 공고 비교
- 지원현황 관리
- 사용자 인증(기본 로그인/회원가입 또는 임시 사용자 세션)

## 5. 확장 기능
- 실제 채용 공고 크롤링 연동
- 다양한 채용 사이트 지원
- 사용자 맞춤형 추천 기능
- 지원 일정 알림 기능
- 공고 북마크 및 필터링 기능
- 분석 결과 저장 및 히스토리 관리
- 기업/직무별 통계 시각화
